парсер-для работы с API сайтов hh и superjob с вакансиями


Платформы для сбора вакансий
hh.ru (ссылка на API: https://github.com/hhru/api)
superjob.ru (ссылка на API: https://api.superjob.ru/)
для работы скачать из репозитория https://github.com/Timerllan/parsers_hh_superjob/tree/main/hh_superjob_api
произвести настройку для полноценной работы нужно https://api.superjob.ru/ произвести регистрацию на данном сайте
и получить ключь Api для Windows - зайти в "настройки",после перейти в заздел "о системе",зайти в "дополнительные параметры системы",
и в "переменные системы" установить токен и назвать переменную "Api_sj_key", после получения данных при сохранении после
названия укажите .json
